# 휴넷 회의실 동기화 크롬 확장 프로그램

다우오피스에서 회의실을 예약·수정·취소하면 **휴넷 면접 일정 시스템에 자동으로 반영**되는 크롬 확장 프로그램입니다.

---

## 배경

휴넷은 사내 그룹웨어로 **다우오피스**를 사용하여 회의실을 예약합니다. 하지만 다우오피스는 외부 API를 제공하지 않기 때문에, 면접 일정 시스템과 직접 연동이 불가능했습니다. 기존에는 다우오피스에서 예약한 뒤 면접 일정 시스템에도 동일한 내용을 수동으로 입력해야 했습니다.

이 확장 프로그램은 다우오피스 페이지에서 발생하는 예약 관련 네트워크 요청을 감지하여, 면접 일정 시스템에 자동으로 동기화합니다.

---

## 동작 원리

```
다우오피스에서 회의실 예약 / 수정 / 취소
        ↓
확장 프로그램이 네트워크 요청 자동 감지
        ↓
면접 일정 시스템 API 호출
        ↓
캘린더에 자동 반영
```

별도로 버튼을 누르거나 내보내기를 할 필요 없이, 다우오피스를 평소처럼 사용하면 됩니다.

---

## 지원 동작

| 다우오피스 행동 | 면접 일정 시스템 반영 |
|---|---|
| 회의실 예약 | 캘린더에 예약 추가 |
| 회의실 예약 수정 | 캘린더 예약 정보 업데이트 |
| 회의실 예약 취소 | 캘린더에서 예약 삭제 |

---

## 사전 준비

### 1. 면접 일정 시스템 환경변수 설정

확장 프로그램이 시스템 API를 호출할 때 사용하는 인증 키를 등록해야 합니다.

**로컬 개발 환경** (`.env.local`):
```
ROOM_SYNC_API_KEY=원하는-비밀키-직접-설정
```

**Vercel 운영 환경**:
Vercel 대시보드 → Settings → Environment Variables에서 동일하게 등록합니다.

> 비밀키는 영문, 숫자 조합으로 자유롭게 설정하세요. 예: `hunet-room-sync-2026`

## 설치 방법

### 1. 코드 받기

```bash
git clone https://github.com/[계정명]/hunet-room-sync.git
```

또는 GitHub에서 ZIP으로 다운로드 후 압축 해제.

### 2. 크롬에 설치

1. 크롬 주소창에 `chrome://extensions` 입력 후 이동
2. 우측 상단 **"개발자 모드"** 토글 켜기
3. **"압축 해제된 확장 프로그램 로드"** 버튼 클릭
4. 다운받은 `hunet-room-sync` 폴더 선택
5. 확장 프로그램 목록에 **"휴넷 회의실 동기화"** 가 나타나면 설치 완료

---

## 초기 설정

설치 후 반드시 한 번 설정을 완료해야 합니다.

1. 크롬 우측 상단 확장 프로그램 아이콘(퍼즐 모양) 클릭
2. **"휴넷 회의실 동기화"** 옆 핀 아이콘을 눌러 고정 (선택사항)
3. 확장 프로그램 아이콘 클릭 → 설정 팝업 열기
4. 아래 두 가지 입력 후 **저장** 클릭

| 항목 | 입력값 예시 |
|---|---|
| 시스템 URL | `https://hunet-interview-scheduler.vercel.app` |
| API 키 | `.env.local`에 설정한 `ROOM_SYNC_API_KEY` 값 |

---

## 사용 방법

설정 완료 후에는 **별도의 조작이 필요 없습니다.**

1. 크롬에서 다우오피스(hug.hunet.co.kr) 접속 및 로그인
2. 평소처럼 회의실 예약 / 수정 / 취소
3. 면접 일정 시스템 캘린더에서 자동 반영 확인

> 확장 프로그램이 동작하려면 다우오피스가 **같은 크롬 브라우저**에서 열려 있어야 합니다.

---

## 다른 컴퓨터에서 사용하기

1. 위 **설치 방법**을 동일하게 진행합니다.
2. **초기 설정**에서 시스템 URL과 API 키를 동일하게 입력합니다.

---

## 주의사항

- 이 확장 프로그램은 `hug.hunet.co.kr` 도메인에서만 동작합니다.
- 다우오피스 로그인은 직접 해야 합니다. 확장 프로그램이 로그인 정보를 저장하거나 전송하지 않습니다.
- 다우오피스의 내부 구조가 변경될 경우 동기화가 동작하지 않을 수 있습니다. 이상 발생 시 개발 담당자에게 문의하세요.
